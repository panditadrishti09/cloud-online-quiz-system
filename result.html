☁️ Cloud Computing Concepts Used

SaaS PaaS IaaS HDFS
  • SaaS: This web quiz app is delivered as a service via browser
  • PaaS: Node.js + Express provides the runtime platform
  • IaaS: App deploys on AWS EC2 (virtual machine) in production
  • HDFS: Quiz attempt logs are stored in Hadoop HDFS for analysis
  • MapReduce: Log files processed to count attempts, errors, high scores
  • MongoDB Atlas: Cloud database (PaaS) stores users, questions, scores
